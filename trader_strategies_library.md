**典型行为**:
```

持仓时间: 2-4 周
胜率: 60-70%
平均收益: +40%
```

**可复制策略**:
1. 监控 RealClearPolitics 等民调聚合网站
2. 对比 Polymarket 价格与民调数据
3. 当偏差 >5% 时押注
4. 选举前3天平仓（避免黑天鹅）

**关键指标**:
- 民调 vs 市场定价差异
- 早期投票数据
- 选民登记变化
- 博彩资金流向

胜率: 65-75%

**可复制策略**:
1. 监控 Twitter/Reddit 伤病消息
2. 对比 Vegas 赔率与 Polymarket
3. 信息优势 >30分钟时入场

### 模式4: 长尾事件捕手

**特征识别**:
- 交易低流动性市场
- 押注极端事件
- 长期持有（数月）

**典型行为**:
```
交易频率: 5-10 笔/月
持仓时间: 1-6 个月
策略: 黑天鹅事件对冲
胜率: 10-20%
平均收益: +300%（获胜时）
```

**可复制策略**:
1. 寻找定价 <5% 的极端事件
2. 基本面分析事件可能性
3. 小资金分散押注（$10-20/个）
4. 持有到事件结果

**例子**:
- "Trump 被弹劾" (定价 5%，实际概率可能 15%)
- "Bitcoin 跌破 $10k" (定价 8%，实际概率可能 20%)
- "外星人被证实存在" (定价 1%，实际概率可能 3%)

### 策略1: 政治民调偏差套利 (PBDA)

**核心逻辑**:
```
当 民调预测 ≠ 市场定价 时，
且偏差 > 历史平均误差，
押注市场错误定价的一方。
```

**实施步骤**:
1. 收集民调数据 (538, RCP)
2. 标准化处理 (加权平均)

### 策略2: 体育信息套利 (SIA)

### 策略3: 波动率做市 (VMM)

**核心逻辑**:
```
在高波动事件前后提供流动性，
赚取买卖价差 + 波动率溢价。
```

**事件类型**:
- 选举结果公布夜
- Fed 利率决议公布
- 重要经济数据发布
- 体育决赛

**实施步骤**:
1. 识别高波动事件日历
2. 事件前2小时扩大价差
3. 事件后快速调整报价
4. 利用恐慌/贪婪情绪
5. 24小时内平仓
